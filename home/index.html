🏁 Pit Stop — Getting Started

OVERVIEW

🎯 What Is This Challenge?

You'll race through 4 enterprise software challenges using GitHub Copilot as your AI pair programmer. Each challenge focuses on a critical pillar of production-ready software: documentation, testing, CI/CD, and exception handling.

Your mission is to apply Copilot's AI capabilities to transform the Snake Cash Rush game project into enterprise-grade code — one challenge at a time.

🧰 Tools We'll Be Using

  • Visual Studio Code — Our code editor
  • GitHub Copilot — AI pair programmer (Chat, Inline, and Agent mode)
  • GitHub Copilot Chat — Ask questions, generate code, and get explanations
  • Python 3.10+ — The Snake game backend runs on Python
  • A modern browser — Microsoft Edge or Chrome to run the game

⚙️ Setting Up GitHub Copilot in VS Code

  1. Open VS Code and go to the Extensions panel (Ctrl+Shift+X)
  2. Search for "GitHub Copilot" and install the extension
  3. Also install "GitHub Copilot Chat" for the chat panel
  4. Sign in with your GitHub account when prompted
  5. Verify Copilot is active — look for the Copilot icon in the status bar
  6. Open Copilot Chat with Ctrl+Shift+I or click the chat icon in the sidebar

📋 Pre-requisites for the Snake Cash Rush Project

  • Visual Studio Code — Download VS Code
  • Python 3.10 or later — Download Python

    👆 Click the yellow button on the Python downloads page

  • A modern browser (Edge or Chrome)
  • Internet access — PyScript and Google Fonts are fetched from CDNs on first load
▶️ Running the Snake Game (optional)
